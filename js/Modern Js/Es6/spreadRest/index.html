Rest and Spread in java script

Rest

The rest operator in javascript allows you to gather the remaining elements of an array object into a new array or object. it's particularly useful when you want to work with a variable number of arguments in functions or to extract some parts of an array or object while keeping the rest.

Spread

The spread operator in javascript allows you to spread out elements of an iterable (like an array or a string) or properties of an object. It is represented by three dots(...). The spread operator is particularly usefull for array and object manipulation, such as merging,copying, or adding elements.
