the rest operator in javascript allows you to gather the remaining elements of an array or object into a new array or object. It's particularly useful when you want to work with a variable number of arguments in functions or to extract some parts of an array or object while keeping the rest


spread operator in javascript allows you to spread out elements of an iterable (like an array or a string) or properties of an object. it is represented by three dots (...). The spread operator is particularly usefull for array anf object manipulation such as copying, merging, or adding elements.
